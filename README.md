# Nail'art Studio

Site web pour salon de beauté avec système de réservation.

## Installation

```bash
npm install
npm run dev
```

Le site sera disponible sur http://localhost:3000

## Configuration

Copiez `.env.local.example` en `.env.local` et configurez les variables d'environnement nécessaires.

## Déploiement

Configurez les variables d'environnement dans votre plateforme (Vercel, etc.) avant le déploiement.
